Physical Design Legend (Task 3B)

  • PK = Primary Key  |  FK = Foreign Key  |  PFK = Primary + Foreign Key
  • Clustered = B-Tree Clustered Index on composite fact table keys
  • Unique B-Tree = B-Tree Index on high-selectivity dimension keys
  • Bitmap = Bitmap Index on low-selectivity columns (ZONE, URBAN, SALE)
  • Partitioning: FactWeeklySales horizontally partitioned by Category (24 partitions)
  • SCD Type 1: Slowly Changing Dimension — overwrite on change
